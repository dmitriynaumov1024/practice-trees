# Practice: Trees

A simple web application to view and edit tree structures.

## Technologies

- Database: json files for sake of simplicity
- Backend: node + express
- Frontend: vue

## Prerequisites

- Windows or Linux or MacOS 
- node >= 18
- npm >= 8
